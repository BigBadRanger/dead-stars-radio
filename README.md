
# Dead Stars Radio

Dead Stars Radio is an AI-assisted web application that transforms real stellar light curve data into sound.

Users tune into stars as if they were radio stations. Brightness over time becomes pitch, rhythm, and texture. AI narration explains what the listener is hearing and whether the signal represents a living star or a fossil of something already gone.

## Why I Built It

Most AI demos answer questions. I wanted to build an experience that used AI, scientific data, sonification, and storytelling to make astronomy emotionally understandable.

## Features

- NASA light curve data
- Stellar sonification
- AI-generated narration
- Multi-spectrum visual treatment
- Retro mission-console interface
- Educational narrative layer

## Architecture

Tech stack
1.	React (frontend) 
2.	Tone.js (audio synthesis) 
3.	NASA MAST API 
4.	NASA Exoplanet Archive 
5.	Claude API (narration agent) 
6.	Stable Diffusion / DALL-E (star imagery) D3.js (waveform viz) 
7.	Python backend (data pre-processing)
